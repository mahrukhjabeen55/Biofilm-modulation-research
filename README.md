# HEAL-4WARD: Biofilm Modulation Research

This repository documents my research on the role of polymeric nanostructures in modulating multispecies biofilms (e.g., *S. aureus*, *P. aeruginosa*, *B. fragilis*) in chronic wounds. My project integrates nanomaterials synthesis, microbiology, and computational predictive modeling to develop next-generation wound dressings.

## Technical Infrastructure & Characterization
My research utilizes high-end departmental and university infrastructure to fingerprint material performance:
- **Nanostructure Synthesis & Characterization:** SAXS (SAXSLab Sapienza), FTIR & microFTIR, NMR (Bruker Avance NEO 400), DSC, TGA, and AFM (PESCA).
- **Microbiological Assays:** Aerobic and anaerobic biofilm cultivation, Confocal Laser Scanning Microscopy (CLSM) coupled with FISH for spatial organization analysis.

## Computational Pipeline
I employ a data-driven approach to map material properties to biological outcomes:
- **Data Integration:** Automated fingerprinting of raw instrument data into a standardized CSV schema.
- **Predictive Modeling:** Use of Python and Machine Learning (Random Forest) to predict biofilm disruption efficacy based on material parameters (Zeta potential, thermal stability, carbonyl density).

## Repository Contents
- `HEAL_4WARD_Master_Schema.csv`: The standardized dataset linking material characterization to biological assay outcomes.
- `analysis_pipeline.py`: The Python script used for processing instrument data and predictive modeling.
